# Zeno Launch Studio UI

Professional interactive frontend for Hackathon Problem Statement 2:

`One-stop product launch studio - from a photo to marketplace-ready in minutes`

## What This UI Demonstrates

- Product photo upload flow
- AI pipeline status visualization
- Marketplace-ready listing output cards
- Product intelligence, compliance, pricing, and packaging sections
- Mobile-responsive, presentation-ready design

## Run Locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in terminal.

## Notes

- This is a frontend prototype with realistic mock generation behavior.
- Hook these UI events to your FastAPI backend endpoints when your services are ready.
